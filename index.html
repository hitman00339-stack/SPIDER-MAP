🔧 GPS Not Working? Read This
Problem: File opened from Downloads folder
Local files (file://) cannot use GPS on Android/iPhone.

Fix Option 1 — Use Google Maps for coordinates:
1. Open Google Maps
2. Walk to your tree
3. Long-press on map at your location
4. Coordinates appear at top (like 28.9456, 77.1015)
5. Tap them → Copy
6. Paste into Lat/Lon fields in Tree tab

Fix Option 2 — Upload file to Google Drive:
1. Upload spider_map_v6.html to Google Drive
2. Open Drive on phone
3. Tap file → Open with Chrome
4. GPS works because it runs from a server URL

Fix Option 3 — Use GitHub Pages (free):
Upload file to github.com → enable Pages → open the https:// link → GPS works perfectly
